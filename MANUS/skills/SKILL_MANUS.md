# MANUS — СКИЛЛЫ

## РОЛЬ В ЭКОСИСТЕМЕ
Manus — автономный агент для сложных исследовательских и многоэтапных задач.

## СИЛЬНЫЕ СТОРОНЫ
- Автономное выполнение задач от А до Я
- Веб-исследования и анализ рынка
- Создание документов и отчётов
- Работа с файлами и данными
- Многоэтапные задачи без ручного контроля

## ЗАДАЧИ
1. **Исследование рынка** — анализ ниш, конкурентов, трендов
2. **Подбор продуктов** — поиск winning products для TeemDrop
3. **Аналитика** — сбор данных, создание отчётов
4. **Документация** — создание бизнес-планов, стратегий
5. **Комплексные задачи** — всё что требует 5+ шагов

## ШАБЛОНЫ ЗАДАЧ

### Исследование ниши
```
Исследуй нишу: [название]
1. Размер рынка и тренды
2. Топ-10 конкурентов (Shopify/Etsy)
3. Ценовой диапазон
4. Лучшие ключевые слова (Pinterest, Google)
5. Рекомендации: входить или нет, почему
Сохрани отчёт в AI_BRAIN/MANUS/outputs/
```

### Поиск продуктов
```
Найди 10 trending продуктов для print-on-demand:
Источники: Etsy, Amazon, Pinterest trends
Критерии: маржа >$12, стабильный спрос, низкая конкуренция
Для каждого: название, цена рынка, примерная себестоимость, целевая аудитория
```
