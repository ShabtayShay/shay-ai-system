# Presentation Builder for PROPAGANDR.AI

## Brand Colors
- Background: #0A0A0A (black)
- Accent: #FF3C00 (orange-red)
- Secondary: #00D4FF (electric blue)
- Text: #F5F0E8 (cream)

## Typography
- Titles: Bebas Neue (bold, condensed)
- Body: Montserrat (clean, modern)

## Style Principles
Bold. Minimal. Provocative. Like the brand.
Each slide = 1 idea. No fluff.

## Slide Structure
1. Hook slide — one powerful statement
2. Problem slide — what's broken in the world
3. Solution slide — PROPAGANDR.AI approach
4. Proof slide — numbers, results, system
5. CTA slide — what to do next
