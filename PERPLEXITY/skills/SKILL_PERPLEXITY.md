# PERPLEXITY — СКИЛЛЫ

## РОЛЬ В ЭКОСИСТЕМЕ
Perplexity — поисковый ИИ для реалтайм исследований и трендов.

## СИЛЬНЫЕ СТОРОНЫ
- Актуальные данные из интернета в реальном времени
- Поиск с источниками и ссылками
- Анализ трендов
- Конкурентная разведка
- Быстрые ответы на фактические вопросы

## ЗАДАЧИ
1. **Тренды** — что сейчас продаётся, вирусится, растёт
2. **Исследование ниш** — спрос, конкуренция, сезонность
3. **Конкуренты** — цены, стратегии, отзывы
4. **SEO** — ключевые слова, вопросы аудитории
5. **Новости** — изменения в Shopify, TikTok, Pinterest алгоритмах

## ШАБЛОНЫ ЗАПРОСОВ

### Тренды продуктов
```

```

