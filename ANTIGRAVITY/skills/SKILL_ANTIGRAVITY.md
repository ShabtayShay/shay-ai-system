# ANTIGRAVITY — СКИЛЛЫ

## РОЛЬ В ЭКОСИСТЕМЕ
Antigravity — AI инструмент для креатива, дизайна и визуального контента.

## СИЛЬНЫЕ СТОРОНЫ
- Генерация изображений и дизайнов
- Креативные концепции
- Визуальный брендинг
- Обработка и стилизация изображений
- Быстрое прототипирование визуалов

## ЗАДАЧИ
1. **Дизайн принтов** — футболки, кружки, худи для TeemDrop
2. **Pinterest визуалы** — пины, обложки, инфографика
3. **Брендинг** — логотипы, цветовые палитры, стиль
4. **Mockups** — визуализация продуктов на моделях
5. **Контент-визуалы** — обложки для видео, сториз

## ШАБЛОНЫ

### Дизайн принта для футболки
```
Создай принт для футболки:
Тема: [тема]
Стиль: [минимализм/винтаж/мем/типографика]
Цвета: [максимум 3-4 цвета]
Размер: 4500x5400px, 300 DPI
Формат: PNG с прозрачным фоном
```

### Pinterest pin дизайн
```
Создай Pinterest pin:
Размер: 1000x1500px
Тема: [тема]
Стиль: чистый, привлекательный
Текст на изображении: [заголовок]
CTA: "Shop Now" / "Learn More"
```
