# CHATGPT — БИБЛИОТЕКА ПРОМПТОВ

## 1. ОПИСАНИЕ ПРОДУКТА (Shopify)
```
Ты — копирайтер для Shopify магазина. Напиши продающее описание:
Продукт: [название]
Тип: [футболка/кружка/худи]
Целевая аудитория: [описание]
Тон: дружелюбный, с юмором
Формат: заголовок (до 60 символов) + 3 буллета + описание (100-150 слов) + CTA
Включи SEO ключевые слова: [ключевики]
```

## 2. СКРИПТ TIKTOK/REELS
```
Напиши скрипт для вертикального видео:
Тема: [тема]
Длина: [15/30/60] секунд
Структура: ХУК (1-3 сек) → ПРОБЛЕМА → РЕШЕНИЕ → CTA
Тон: [энергичный/саркастичный/информативный]
Добавь визуальные указания для монтажа [в скобках]
```

## 3. PINTEREST PIN
```
Создай текст для Pinterest pin:
Продукт/тема: [описание]
Заголовок пина: до 100 символов, SEO
Описание: 150-200 символов с ключевыми словами
Alt text: описание изображения для SEO
Доска: [название доски]
```

## 4. EMAIL РАССЫЛКА
```
Напиши email для: [welcome/abandoned cart/promotion]
Бренд: [название магазина]
Тон: дружелюбный, не спамный
Структура: тема письма (до 50 симв) + превью + тело + CTA кнопка
Цель: [возврат/покупка/подписка]
```

## 5. BRAINSTORM НИШИ
```
Предложи 10 нишевых идей для print-on-demand:
Платформа: Shopify + TeemDrop
Критерии: высокий спрос, низкая конкуренция, маржа >$12
Для каждой идеи: название ниши, целевая аудитория, 3 примера продуктов, почему это сработает
```

## 6. КОНТЕНТ-ПЛАН
```
Создай контент-план на неделю:
Платформы: TikTok, Pinterest, Instagram
Ниша: [ниша]
Частота: [X] постов/день на платформу
Формат: таблица | День | Платформа | Тема | Формат | Хук |
```
