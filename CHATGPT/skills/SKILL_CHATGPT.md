# CHATGPT — СКИЛЛЫ

## РОЛЬ В ЭКОСИСТЕМЕ
ChatGPT — креативный генератор и текстовый движок Shay.

## СИЛЬНЫЕ СТОРОНЫ
- Генерация длинных текстов (блоги, статьи, описания)
- Креативный копирайтинг
- Brainstorm идей
- Перевод и адаптация контента (RU/HE/EN)
- Создание GPTs для повторяющихся задач

## ЗАДАЧИ
1. **Копирайтинг** — описания продуктов Shopify, email рассылки
2. **Скрипты видео** — TikTok, Reels сценарии
3. **Переводы** — адаптация контента на 3 языка
4. **Brainstorm** — идеи продуктов, ниш, контента
5. **GPTs** — кастомные ассистенты для повторяющихся задач

## КАСТОМНЫЕ GPTs
- **Product Writer** — описания продуктов для Shopify
- **Content Planner** — контент-план на неделю/месяц
- **Pin Creator** — описания для Pinterest пинов
- **Script Writer** — скрипты для коротких видео

## ПРОМПТ-ШАБЛОНЫ
Хранятся в: `AI_BRAIN/CHATGPT/prompts/`

## ИНТЕГРАЦИЯ
- ChatGPT API через n8n для авто-генерации
- Результаты → CHATGPT/outputs/
- Уведомление в Telegram после генерации
