# Social Traffic Skill

Manage TikTok, Pinterest, YouTube Shorts, and Reels publishing via Blotato API.

## Context

- Blotato API handles social media publishing
- Pinterest: connected (board: mindovertoxicity)
- TikTok: requires media attachment (text-only not supported)
- Traffic flow: TeemDrop product → Shopify → Social → Customer orders → Fulfillment

## Instructions

When working with social media publishing:
1. TikTok posts MUST include a video/image — text-only is not supported
2. Pinterest pins should target the correct board
3. Use short-form video format for TikTok/Reels/Shorts
4. Include product links and CTAs in descriptions
5. Track engagement metrics to optimize posting schedule
