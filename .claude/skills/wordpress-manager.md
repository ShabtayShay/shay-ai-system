# WordPress Manager Skill

Manage the 10 WordPress sites for automated content publishing.

## Context

- All 10 WordPress sites hosted on VPS2 (72.60.81.215)
- Content published via n8n workflow using WordPress REST API
- Sites are part of the automated income system

## Instructions

When working with WordPress sites:
1. Use WordPress REST API v2 for all interactions
2. Authenticate via application passwords (never store credentials in code)
3. Always set proper categories and tags for SEO
4. Include featured images when available
5. Schedule posts at optimal times for traffic
