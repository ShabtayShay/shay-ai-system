# Dropshipping Automation Skill

Manage the TeemDrop dropshipping automation pipeline.

## Context

- Product source: TeemDrop
- Store: Shopify
- Traffic: TikTok, Reels, Shorts, Pinterest
- Flow: TeemDrop product → Shopify → Social traffic → Customer orders → TeemDrop fulfillment → Profit

## Instructions

When working with the dropshipping pipeline:
1. Product listings must include compelling descriptions and media
2. Pricing should account for TeemDrop cost + shipping + margin
3. Track conversion rates per traffic source
4. Automate inventory sync between TeemDrop and Shopify
5. Monitor order fulfillment status and notify on delays
