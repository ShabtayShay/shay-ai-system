# EIGENT — СКИЛЛЫ

## РОЛЬ В ЭКОСИСТЕМЕ
Eigent — платформа AI-агентов для автономного выполнения задач.

## СИЛЬНЫЕ СТОРОНЫ
- Автономные агенты с веб-доступом
- Многоэтапные задачи без контроля
- Работа с браузером (клики, формы, навигация)
- Интеграция с внешними сервисами
- Параллельное выполнение задач

## ЗАДАЧИ
1. **Веб-агенты** — регистрация, заполнение форм, мониторинг
2. **Скрапинг** — сбор данных с сайтов конкурентов
3. **Мониторинг** — отслеживание цен, наличия, изменений
4. **Автоматизация** — рутинные задачи в браузере
5. **Тестирование** — проверка Shopify магазина, воронки

## ПРИМЕНЕНИЕ

### Мониторинг конкурентов
```
Агент: каждый день проверяй [URL конкурента]
Следи за: новые продукты, изменение цен, промо-акции
Отчёт → AI_BRAIN/EIGENT/outputs/competitor_YYYY-MM-DD.md
```

### Проверка магазина
```
Агент: пройди как покупатель через весь flow:
1. Открой магазин
2. Найди продукт
3. Добавь в корзину
4. Проверь checkout
5. Отчёт: что работает, что сломано
```

## ВЫХОДНЫЕ ФАЙЛЫ
- Агенты (конфиги): `AI_BRAIN/EIGENT/agents/`
- Результаты: `AI_BRAIN/EIGENT/outputs/`
